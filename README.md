# Installation

* Clone the repository.
``` 
 $ git clone git@github.com:Anuj-Gupta79/DashBoard-UI.git
``` 

* Navigate to the project directory.
``` 
 $ cd dashboard
```

* Install dependencies
```
 $ npm install
```

* Start the application
```
 $ npm start
```

# Usage

* Open your browser and go to http://localhost:3000
